CSS Style Guide

HTML

• Font ‘Oswald (WebFont)’ with normal weight and style, this is for readability as well as aesthetic quality.
• Background color set to light red (#FFE8b7) to provide a light pastel background that is easy on the eyes and makes people excited about the sports.
• Text aligned to the center generically, as it is readable from users.

Body

• Font is Arial with size of 14px and line-height 20px
• Color is #333 which is light yellow.

Typography

• "Source Sans Pro" is the font used.
• Headers 1 to 3 with corresponding properties and values in descending sizes and weights. E.g. ( font-size 32px and 24 px respectively). And margin-top:20px.
• Identified bold elements with corresponding CSS style.

Textlogo

• Has center text-align and text-shadow with #333 2px 2px and 0px.
• Color is light red #ffe8b7 with font size 31px (sans-serif).
• Line-height and font-weight is 31px and 700 respectively.

Navigation

• Horizontal bar relatively positioned with padding-top and bottom 10px.
• Bars with white background color #fd4a4a, to stand out against the background color.

Button

• Heading text aligned to the centre with font-size 15px and weight 700.
• Padding is using 9px 15px;
• Using background color light yellow #3898ec.
• The article itself has a margin-left, margin-right and margin-top of auto, auto and 40px respectively. Box-shadow using rgba(13,1,1,0.49) 2px 2px 0px 0px.

Images

• Photograph is displayed as inline- block element. With max-width is 100%.
• Margin-left and margins-right set to auto.
• Logo image floating to the left with 0 padding and margin. It also has shadow effect with #333 2px 2px 0px. And padding top with 40px.

Header

• Normally, header using color light red (#ffe8b7) and font-style italic.
• Text-shadow is using #1d0606 1px 1px 0px and (sans-serif).
• The text-align is centered and with font-weight 600.

labels

• In the feedback form, the logo is centered as a block element, and links to the homepage.
• Displayed as block and margin bottom 5px;

Input

• Displayed as block with width and height 100% and 38px respectively.
• Background color is #ffffff and color using #333333
• Padding and margin-bottom is 8px 12px and 10px.

Footer

• Has a font size of 16 px, font ‘sans-serif’ and the light red color #ffe8b7. With padding 25px 10px and 10px and margin auto.
• Copyright healthy.me 2015 is placed as text in the footer of every page.
