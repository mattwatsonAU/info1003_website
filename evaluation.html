Website Evaluation Plan

1. Does the website provide the user with ease of access to the information they are looking for?

2. Is the user able to understand all the phrases and content displayed on the website?

3. Is the user able to navigate the website easily and efficiently?

4. Are the each of the webpages consistent in form, style and structure?

5. Does the user find the design appropriate and aesthetic?

6. Does the website employ error prevention techniques?

7. Are users able to overcome any issues with the help of the information on the website?

8. Is the help documentation on the website easy to find?
